// ARCHITECTURE

How it works

a two-component injection chain

01
C# Loader
A lightweight WPF launcher finds the javaw.exe process (Lunar Client) and injects bridge.dll via CreateRemoteThread + LoadLibraryA. Hosts system-wide input hooks for keybind dispatch.
‚Üí
02
C++ Bridge DLL
Attaches to the JVM via JVMTI & JNI. Dynamically discovers Minecraft classes at runtime ‚Äî no hardcoded obfuscated names, works across Lunar versions. Hooks SwapBuffers to render overlays every frame.
‚Üí
03
TCP Bridge
Game state (entities, health, screen name) is streamed to the loader at ~20Hz over TCP port 25590. Config changes and commands are pushed back at ~5Hz. Zero packet modification ‚Äî purely read-only game state.
